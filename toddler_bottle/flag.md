**Download [UPX](https://upx.github.io/) and unpack it**:
```
./upx -d flag
```
A new binary will be generated.

**Let see what we can find using `strings` on the unpacked binary**:
```
strings flag | grep ':)'
```

*Remember that the flag is always of a format [words] :) ?*
